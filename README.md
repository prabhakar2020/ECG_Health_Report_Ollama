# ECG_Health_Report
It is automated software system that reads a patient's ECG data and instantly gives a structured heart health report.
Instead of sending sensitive medical ECG data over the internet to expensive cloud services, this project runs 100% locally on your local computer using an open-source AI model (Qwen2.5) and Python.

The system takes raw, messy electrical signals from a heart monitor, cleans the data, calculates critical health metrics (like heart rate and heartbeat consistency), and uses an AI agent to write a clear clinical diagnosis. It can automatically catch conditions like heart beating too fast, heart beating too slow, or regular heart rhythms.
